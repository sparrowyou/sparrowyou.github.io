常量

常量是固定值，在程序执行期间不会改变。这些固定的值，又叫做字面量。
常量可以是任何的基本数据类型，比如整数常量、浮点常量、字符常量，或字符串字面值，也有枚举常量。
常量就像是常规的变量，只不过常量的值在定义后不能进行修改。
常量可以直接在代码中使用，也可以通过定义常量来使用。

  • 整数常量

    整数常量可以是十进制、八进制或十六进制的常量。前缀指定基数：0x 或 0X 表> 示十六进制，0 表示八进制，不带前缀则默认表示十进制。
    整数常量也可以带一个后缀，后缀是 U 和 L 的组合，U 表示无符号整数 （unsigned），L 表示长整数（long）。
    后缀可以是大写，也可以是小写，U 和 L 的顺序任意。

C存储类

  • auto

    auto存储类是所有局部变量默认的存储类。
    定义在函数中的变量默认为uto存储类，这意味着它们在函数开始时被创建，在函数结束时被销毁。

    {
    int mount
    auto int mount
}

  • register

    register存储类定义存储在寄存器的局部变量，所以变量的访问速度更快，但是它不能直接取地址，因为它不是存储在RAM中的。
    在需要频繁访问的变量上使用register存储类可以提高程序的运行速度。
    register int variance
    还应注意的是，定义 ‘register’ 并不意味着变量将被存储在寄存器中，它意味着变量可能存储在寄存器中，这取决于硬件和实现的限制。

  • static

    static 存储类指示编译器在程序的生命周期内保持局部变量的存在，而不需要在每次它进入和离开作用域时进行创建和销毁。
    因此，使用 static 修饰局部变量可以在函数调用之间保持局部变量的值。

    static 修饰符也可以应用于全局变量。当 static 修饰全局变量时，会使变量的作用域限制在声明它的文件内。

    全局声明的一个 static 变量或方法可以被任何函数或方法调用，只要这些方法出现在跟 static 变量或方法同一个文件中。

    静态变量在程序中只被初始化一次，即使函数被调用多次，该变量的值也不会重置。

    #include <stdio.h>
 
/* 函数声明 */
void func1(void);

static int count=10;        /* 全局变量 - static 是默认的 */

int main()
{
while (count--) {
    func1();
}
return 0;
}

void func1(void)
{
/* 'thingy' 是 'func1' 的局部变量 - 只初始化一次
* 每次调用函数 'func1' 'thingy' 值不会被重置。
*/                
static int thingy=5;
thingy++;
printf(" thingy 为 %d ， count 为 %d\n", thingy, count);
}

  • extern

    extern 存储类用于定义在其他文件中声明的全局变量或函数。当使用 extern 关键字时，不会为变量分配任何存储空间，
    而只是指示编译器该变量在其他文件中定义。

    extern 存储类用于提供一个全局变量的引用，全局变量对所有的程序文件都是可见的。
    当您使用 extern 时，对于无法初始化的变量，会把变量名指向一个之前定义过的存储位置。

    当您有多个文件且定义了一个可以在其他文件中使用的全局变量或函数时，可以在其他文件中使用 extern 来得到已定义的变量或函数的引用。
    可以这么理解，extern 是用来在另一个文件中声明一个全局变量或函数。

  • 二维数组在内存上的存放方法
    二维数组在逻辑上是方阵，由行和列组成。
    但是二维数组在物理上是线性的，按行来依次进行存放，内存是连续的。
    二维数组名的步长是一行的长度。
    age + 1 address is 00EFFC04
age + 2 address is 00EFFC14

    因为每一行有四个元素，每个int类型的元素占四个字节，一行有16个字节，
    所以数组名age加1后地址增加了16个字节说明数组名的步长位一行的长度。

    具体到每一个元素加1的时候，地址增加的是一个元素所占字节的大小，因此元素的步长即为元素本身的大小。
    age[2][0] + 0 address is 00EFFC14
age[2][0] + 1 address is 00EFFC18
    例子：
    #include <stdio.h>

int main()
{
    int age[6][4];
    for (int i = 0; i < sizeof(age)/sizeof(age[0]) ; i++)
    {
        printf("age + %d address is %p\n",i, age + i);
    }
    for (int i = 0; i < sizeof(age) / sizeof(age[0]); i++)
    {
        for (int j = 0; j < sizeof(age[0]) / sizeof(int); j++)
        {
            printf("age[%d][0] + %d address is %p\n",i,j,&age[i][0]+j);
        }

    }
}

二维数组传递给函数

  1. 第一维的长度可以不指定，但必须指定第二维的长度：
    void print_a(int a[][5], int n, int m)

    编译器是这样处理数组的：
    设有数组 int a[m][n]，如果要访问 a[i][j ]的值，编译器的寻址方式为:

    &a[i][j]=&a[0][0]+i*sizeof(int)*n+j*sizeof(int); // 注意 n 为第二维的维数

  2. 指向一个有5个元素一维数组的指针：
    void print_b(int (*a)[5], int n, int m)

    a是一个指向整数数组的指针，它有5列。因为使用了指针算法，所以可以使用一个单独的指针来访问二维数组中的元素。

    具体来说，(*a)[5]表示一个指向整数数组的指针，该数组有5列。我们可以将其拆分为以下步骤：

    1. (*a)：这将解引用指针，从而获得整数数组的地址。
    2. (*a)[5]：这将解引用整数数组的地址，并表示该数组有5列。
  3. 利用数组是顺序存储的特性,通过降维来访问原数组:
    void print_c(int *a, int n, int m)
    #include <stdio.h>

 void printArray(int *arr, int row, int col)
 {
   for (int i = 0; i < row; i++) 
   {
       for (int j = 0; j < col; j++) 
       {
           printf("%d ", *(arr + i * col + j));
       }
       printf("\n");
   }
 }

 int main() 
 {
   int arr[2][3] = {{1, 2, 3}, {4, 5, 6}};
   printArray(&arr[0][0], 2, 3);
   return 0;
 }

    *(arr + i * col + j)是指针算法，用于访问二维数组中的元素。在这个例子中，arr是一个指向整数的指针，i和j是循环变量，col是列数。因为我们使用了指针算法，所以我们可以使用一个单独的指针来访问二维数组中的元素，而不需要使用两个嵌套的循环。

    具体来说，*(arr + i * col + j)等价于arr[i][j]。我们可以将其拆分为以下步骤：

    1. arr + i * col：这将指针移动到第i行的开头。
    2. *(arr + i * col)：这将解引用指针，从而获得第i行的地址。
    3. *(arr + i * col + j)：这将解引用第i行的地址，并将指针移动到第j列。

指针

指针是一个特殊的变量，它里面存储的数值被解释成为内存里的一个地址。
指针也就是内存地址，指针变量是用来存放内存地址的变量。

形式：type *var_name;

int    *ip;    /* 一个整型的指针 */
double *dp;    /* 一个 double 型的指针 */
float  *fp;    /* 一个浮点型的指针 */
char   *ch;    /* 一个字符型的指针 */

  • null指针
    在变量声明的时候，如果没有确切的地址可以赋值，赋为 NULL 值的指针被称为空指针。

指针的算数运算

C 指针是一个用数值表示的地址。因此可以对指针进行四种算术运算：++、--、+、-。

  • 指针的每一次递增，它其实会指向下一个元素的存储单元。
  • 指针的每一次递减，它都会指向前一个元素的存储单元。
  • 指针在递增和递减时跳跃的字节数取决于指针所指向变量数据类型长度，比如 int 就是 4 个字节。
    列子：
    #include <stdio.h>

const int MAX = 3;

int main ()
{
  int  var[] = {10, 100, 200};
  int  i, *ptr;

  /* 指针中的数组地址 */
  ptr = var;
  for ( i = 0; i < MAX; i++)
  {

      printf("存储地址：var[%d] = %p\n", i, ptr );
      printf("存储值：var[%d] = %d\n", i, *ptr );

      /* 指向下一个位置 */
      ptr++;
  }
  return 0;
}
    指针可以用关系运算符进行比较，如 ==、< 和 >。如果 p1 和 p2 指向两个相关的变量，
    比如同一个数组中的不同元素，则可对 p1 和 p2 进行大小比较。

指针和数组

数组a[ ]的数组名a是一个指向&a[0]的指针，所以可以用指向数组的指针来表示数组：

int a[];
int *p;
p = a;
\* 下面几种表示等价 *\
a[i] == *(a + i);
a[i] == *(p + i);
p[i] == *(p + i);
a[i] == p[i];

例子：
#include <stdio.h>
 
int main ()
{
  /* 带有 5 个元素的整型数组 */
  double balance[5] = {1000.0, 2.0, 3.4, 17.0, 50.0};
  double *p;
  int i;

  p = balance;

  /* 输出数组中每个元素的值 */
  printf( "使用指针的数组值\n");
  for ( i = 0; i < 5; i++ )
  {
      printf("*(p + %d) : %f\n",  i, *(p + i) );
  }

  printf( "使用 balance 作为地址的数组值\n");
  for ( i = 0; i < 5; i++ )
  {
      printf("*(balance + %d) : %f\n",  i, *(balance + i) );
  }

  return 0;
}

指向指针的指针

指向指针的指针是一种多级间接寻址的形式，或者说是一个指针链。通常，一个指针包含一个变量的地址。
当我们定义一个指向指针的指针时，第一个指针包含了第二个指针的地址，第二个指针指向包含实际值的位置。
当一个目标值被一个指针间接指向到另一个指针时，访问这个值需要使用两个星号运算符。

例子：

#include <stdio.h>
 
int main ()
{
    int  V;
    int  *Pt1;
    int  **Pt2;
  
    V = 100;
  
    /* 获取 V 的地址 */
    Pt1 = &V;
  
    /* 使用运算符 & 获取 Pt1 的地址 */
    Pt2 = &Pt1;
  
    /* 使用 pptr 获取值 */
    printf("var = %d\n", V );
    printf("Pt1 = %p\n", Pt1 );
    printf("*Pt1 = %d\n", *Pt1 );
    printf("Pt2 = %p\n", Pt2 );
    printf("**Pt2 = %d\n", **Pt2);
  
    return 0;
}

  • 指针函数
    函数指针是指向函数的指针变量。
    函数指针可以像一般函数一样，用于调用函数、传递参数。
    函数指针变量的声明：
    typedef int (*fun_ptr)(int,int);
    #include <stdio.h>

int add(int a, int b) {
    return a + b;
}

int subtract(int a, int b) {
    return a - b;
}

int multiply(int a, int b) {
    return a * b;
}

int main() {

    operation = subtract;
    printf("%d - %d = %d\n", a, b, operation(a, b));

    operation = multiply;
    printf("%d * %d = %d\n", a, b, operation(a, b));

}
  • 回调函数
    #include<stdio.h>

int Callback_1(int x) // Callback Function 1
{
    printf("Hello, this is Callback_1: x = %d ", x);
    return 0;
}

int Callback_2(int x) // Callback Function 2
{
    printf("Hello, this is Callback_2: x = %d ", x);
    return 0;

{
    printf("Hello, this is Callback_3: x = %d ", x);

int Handle(int y, int (*Callback)(int))
{

}

int main()
{
    int a = 2;
    int b = 4;
    int c = 6;
    printf("Entering Main Function. ");
    Handle(a, Callback_1);
    Handle(b, Callback_2);
    Handle(c, Callback_3);
    printf("Leaving Main Function. ");
    return 0;
}

enum(枚举)

枚举类型通常用于为程序中的一组相关的常量取名字，以便于程序的可读性和维护性。
定义一个枚举类型，需要使用 enum 关键字，后面跟着枚举类型的名称，以及用大括号 {} 括起来的一组枚举常量。
每个枚举常量可以用一个标识符来表示，也可以为它们指定一个整数值，如果没有指定，那么默认从 0 开始递增。

枚举语法定义格式为：
enum　枚举名　{枚举元素1,枚举元素2,……};

枚举变量的定义:
1. 先定义枚举类型，再定义枚举变量
enum DAY
{
      MON=1, TUE, WED, THU, FRI, SAT, SUN
};
enum DAY day;

2. 定义枚举类型的同时定义枚举变量
enum DAY
{
      MON=1, TUE, WED, THU, FRI, SAT, SUN
} day;

3. 省略枚举名称，直接定义枚举变量
enum
{
      MON=1, TUE, WED, THU, FRI, SAT, SUN
} day;

字符串

在 C 语言中，字符串实际上是使用空字符 \0 结尾的一维字符数组。
由于在数组的末尾存储了空字符 \0，所以字符数组的大小比字符串的字符数多一个。
字符串声明：
char str[] = {'h','e','l','l','o',' 'w','o','r','l','d','!'}

s声明时不需要把 null 字符放在字符串常量的末尾。C 编译器会在初始化数组时，自动把 \0 放在字符串的末尾。

结构是 C 编程中一种用户自定义的可用的数据类型，它允许存储不同类型的数据项。
结构体中的数据成员可以是基本数据类型（如 int、float、char 等），也可以是其他结构体类型、指针类型等。

    ...

链表

链表是一种常见的重要的数据结构。它是动态地进行存储分配的一种结构。
链表有一个 头指针变量(如果不提供 头指针 head 则整个链表无法访问。)，它存放一个地址，该地址指向一个元素，链表中每一个元素称为 结点，
每个结点都应包括两个部分，一为用户需要用的实际数据，二为下一个结点的地址。
可以看出，头指针 head 指向第一个元素，第一个元素又指向第二个元素，。。。。
直到最后一个元素，该元素不再指向其他元素，它称为 表尾，它的地址部分放一个 NULL（表示 空地址）链表到此结束。

  • 简单链表示例
    #include <stdio.h>
#include <stdlib.h>
 
#define NULL 0
 
struct student
{
    long num;
    float score;
    struct student *next;
};
 
void main()
{
    struct student a, b, c, *head, *p;
    a.num = 99101; a.score = 89.5;
    b.num = 99103; b.score = 90;
    c.num = 99107; c.score = 85;//对结点的 num 和 score 成员赋值
    head = &a;//将结点 a 的起始地址赋给头指针 head
    a.next = &b;//将结点 b 的起始地址赋给 a 结点的 next 成员
    b.next = &c;
    c.next = NULL;// c 结点的 next 成员不存放其他结点地址
    p = head;//使 p 指针指向 a 结点
    do
    {
        printf("%ld %5.1f\n", p->num, p->score);// 输出 p 指向的结点的数据
        p = p->next;//使 p 指向下一结点
    }while(p != NULL);//输出完 c 结点后 p 的值为 NULL
    system("pause");
}
  • 处理动态链表所需函数
    • malloc函数
      void *malloc(unsigned int size);

      作用是在内存的动态存储区中分配一个长度为size的连接空间。
      些函数的值（即返回值）是一个指向分配空间起始地址的指针（基类型为void）。
      如果些函数未能成功地执行（例如内存空间不足）则返回空指针 NULL。

    • calloc函数
      void *calloc(unsigned n, unsigned size);

      其作用是在内存的动态区存储中分配n个长度为size的连续空间。
      函数返回一个指向分配空间起始地址的指针，如果分配不成功，返回 NULL。
      用calloc函数可以为一维数组开辟动态存储空间，n为数组元素个数，每个元素长度为size。

    • free函数
      void free(void *p);

      其作用是释放由p指向的内存区，使这部分内存区能被其它变量使用，
      p是最后一次调用calloc或 malloc函数时返回的值。free函数无返回值。

  • 动态链表示例
    #include <stdio.h> // 引入标准输入输出头文件，用于输入输出函数
#include <stdlib.h> // 引入标准库头文件，用于内存分配和系统调用函数

#define NULL 0 // 定义一个宏，表示空指针
#define LEN sizeof(struct student) // 定义一个宏，表示学生结构体的大小

struct student // 定义一个结构体类型，名为student
{
    long num; // 定义一个长整型变量，表示学号
    float score; // 定义一个浮点型变量，表示成绩
    struct student *next; // 定义一个指向student类型的指针，表示下一个学生
};

struct student *create() // 定义一个函数，返回值是指向student类型的指针，函数名为create
{
    struct student *p1, *p2, *head; // 声明三个指向student类型的指针，分别为p1,p2和head
    int num; // 声明一个整型变量，表示学号
    float score; // 声明一个浮点型变量，表示成绩
    int n = 0; // 声明并初始化一个整型变量，表示学生的个数

    head = NULL; // 将head指针初始化为空

    p1 = p2 = (struct student *)malloc(LEN); // 为p1和p2指针分配一块大小为LEN的内存空间，并强制转换为student类型的指针

    printf("please input num and score.\n"); // 打印提示信息，让用户输入学号和成绩
    scanf("%d,%f", &p1->num, &p1->score); // 从标准输入读取两个值，并赋给p1所指向的结构体的num和score成员

    while(p1->num != 0) // 当输入的学号不为0时，进入循环
    {
        n ++; // 学生个数加一
        if(n == 1) // 如果是第一个学生
            head = p1; // 将head指针指向p1所指向的结构体
        else // 如果不是第一个学生
            p2->next = p1; // 将p2所指向的结构体的next成员指向p1所指向的结构体
        p2 = p1; // 将p2指针指向p1所指向的结构体
        p1 = (struct student *)malloc(sizeof(struct student)); // 为p1指针重新分配一块大小为LEN的内存空间，并强制转换为student类型的指针

        printf("please input num and score.\n"); // 打印提示信息，让用户继续输入学号和成绩

        scanf("%d,%f", &p1->num, &p1->score); // 从标准输入读取两个值，并赋给p1所指向的结构体的num和score成员
    }
    p2->next = NULL; // 当输入的学号为0时，退出循环，并将p2所指向的结构体的next成员设为空
    return head; // 返回head指针，即链表的头节点
}

void printlist(struct student *head) // 定义一个函数，无返回值，参数是指向student类型的指针，函数名为printlist
{
    struct student *p; // 声明一个指向student类型的指针，名为p
    p = head; // 将p指针指向head所指向的结构体，即链表的头节点
    if(head != NULL) // 如果链表不为空
    {
        do
        {
            printf("num=%d score=%f\n", p->num, p->score); // 打印p所指向的结构体的num和score成员的值
            p = p->next; // 将p指针指向p所指向的结构体的next成员所指向的结构体，即链表的下一个节点
        }while(p != NULL); // 当p不为空时，重复上述步骤
    }
}

void main() // 定义主函数，无返回值，函数名为main
{
    struct student *head; // 声明一个指向student类型的指针，名为head
    head = create(); // 调用create函数，将返回值赋给head指针，即创建一个链表
    printlist(head); // 调用printlist函数，传入head指针作为参数，即打印链表中的每个学生的信息
    system("pause"); // 调用system函数，传入"pause"作为参数，即暂停程序的执行，等待用户输入
}

    /* 定义一个函数，返回值是指向student类型的指针，函数名为delNode，
参数是指向student类型的指针head和一个整型变量num */
struct student *delNode(struct student *head, int num) 
{
    printf("delNode.\n"); // 打印提示信息，表示进入了delNode函数
    struct student *p1, *p2; // 声明两个指向student类型的指针，分别为p1和p2
    if(head == NULL) // 如果链表为空，即head指针为空
    {
        printf("The List is NULL.\n"); // 打印提示信息，表示链表为空
    }
    else // 如果链表不为空
    {
        p1 = head; // 将p1指针指向head所指向的结构体，即链表的头节点
        /* 当p1所指向的结构体的next成员不为空，并且p1所指向的
        结构体的num成员不等于要删除的学号时，进入循环 */
        while(p1->next != NULL && p1->num != num) 
        {
            p2 = p1; // 将p2指针指向p1所指向的结构体
            p1 = p1->next; // 将p1指针指向p1所指向的结构体的next成员所指向的结构体，即链表的下一个节点
        }
        /* 当退出循环后，如果p1所指向的结构体的num成员等于要删除的
        学号，说明找到了要删除的节点 */
        if(p1->num == num) 
        {
            if(p1 == head) // 如果要删除的节点是头节点
                /* 将head指针指向p1所指向的结构体的next成员
                所指向的结构体，即链表的第二个节点 */
                head = p1->next; 
            else // 如果要删除的节点不是头节点
                /* 将p2所指向的结构体的next成员指向p1所指向的结构体的next成员所指向的结构体，
                即跳过要删除的节点，将前后两个节点连接起来 */
                p2->next = p1->next; 
        }
        /* 如果退出循环后，如果p1所指向的结构体的num成员不等于要
        删除的学号，说明没有找到要删除的节点 */
        else 
            printf("Can not find list num.\n"); // 打印提示信息，表示无法找到要删除的学号
    }
    return head; // 返回head指针，即删除后的链表的头节点
}

共用体

共用体是C语言中一种特殊的数据类型，它可以让几个不同类型的变量共享同一段内存空间。
共用体的定义格式和结构体类似，但是共用体的成员只能同时使用一个，而结构体的成员可以同时使用多个。
共用体的作用是节省内存空间，或者实现不同类型数据之间的转换。
例子：

union [union tag]
{
   member definition;
   member definition;
   ...
   member definition;
} [one or more union variables];

共用体变量所占的内存长度等于最长的成员变量的长度。

位域

位域是一种数据结构，可以把一个字节中的二进制位划分为不同的区域，
并说明每个区域的位数。位域可以节省存储空间，方便对二进制位进行操作。
定义：

struct 位域结构名 
{

 type [member_name] : width ;

};

使用：

  • 位域变量名.位域名
  • 位域变量名->位域名

    位域的宽度不能超过它所依附的数据类型的长度，例如int型的位域最多只能占32位。
    位域可以是无名的，这样可以用来填充或调整位置。
    位域可以是有符号或无符号的，这会影响它们的取值范围。
    位域的存储方式依赖于编译器，不同的编译器可能有不同的字节对齐规则。
    位域不能跨两个字节，如果剩余的空间不够，就会从下一个字节开始存储。
    位域不能用数组或指针来访问，只能用位域名来操作。

typedef & #define

typedef是一个关键字，用来为已有的数据类型定义一个新的名字。
typedef可以用来简化复杂的类型名称，增加代码的可读性，方便程序的移植和修改。

#define是一个预处理指令，用来定义一个宏123。宏是一个标识符，它与一个常量或一个表达式相关联。
当宏出现在源代码中时，预处理器会用它的值或表达式来替换它。这样可以简化代码，提高效率，避免重复。

区别：

  • typedef是一个关键字，而#define是一个预处理指令。
  • typedef只能为类型定义别名，而#define可以为任何文本定义别名。
  • typedef在编译时进行类型检查，而#define在预处理时进行简单的文本替换，不作正确性检查。
  • typedef定义的别名不能再使用其他类型说明符进行扩展，而#define定义的别名可以。
  • typedef对作用域敏感，而#define对作用域不敏感。

        if (ferror(fp2)) //如果发生错误，跳出循环

            break;

    //关闭目标文件
    fclose(fp2);

}

线性表

线性表是一种数据结构，它是由n个数据特性相同的元素构成的有限序列。
线性表的特点是每个元素有且只有一个前驱和一个后继，除了第一个元素和最后一个元素外。
线性表可以用顺序存储结构或链式存储结构来实现。
顺序存储结构是用一组地址连续的存储单元依次存储线性表的元素，例如用数组来表示线性表。
链式存储结构是用一组任意的存储单元存储线性表的元素，每个元素除了数据域外，还有一个指针域指向下一个元素，例如用链表来表示线性表。

//定义一个最大长度为100的常量
#define MAX_SIZE 100
//定义一个整型数组作为顺序存储结构
int array[MAX_SIZE];
//定义一个变量作为线性表的长度
int length = 0;
//创建一个示例线性表，如下图所示
//array: [10, 20, 30, 40, 50]
//length: 5
void createExampleList() {
  //给数组赋值
  array[0] = 10;
  array[1] = 20;
  array[2] = 30;
  array[3] = 40;
  array[4] = 50;
  //更新长度
  length = 5;
}

在顺序存储结构的线性表中，插入或删除元素需要移动其他元素的位置，以保持线性表的连续性和顺序性。具体的步骤如下：

  • 插入元素
    假设要在第i个位置插入一个元素x，需要先判断i是否合法，即1<=i<=length+1。
    如果不合法，返回错误信息。如果合法，需要先判断线性表是否已满，即length是否等于MAX_SIZE。
    如果已满，返回错误信息。
    如果不满，需要从第length个元素开始，依次将每个元素后移一位，直到第i个元素为止。然后将第i个元素赋值为x，最后将length加一。
    //定义一个插入函数，参数为位置i和元素x
int insert(int i, int x) {
  //判断i是否合法
  if (i < 1 || i > length + 1) {
    //返回错误信息
    printf("Invalid position!\n");
    return -1;
  }
  //判断线性表是否已满
  if (length == MAX_SIZE) {
    //返回错误信息
    printf("List is full!\n");
    return -1;
  }
  //从第length个元素开始，依次后移一位
  for (int j = length; j >= i; j--) {
    array[j] = array[j - 1];
  }
  //将第i个元素赋值为x
  array[i - 1] = x;
  //将length加一
  length++;
  //返回成功信息
  return 0;
}
  • 删除元素
    假设要删除第i个位置的元素，并返回其值，需要先判断i是否合法，即1<=i<=length。
    如果不合法，返回错误信息。
    如果合法，需要先将第i个元素的值保存在一个变量中，然后从第i+1个元素开始，依次将每个元素前移一位，直到第length个元素为止。最后将length减一，并返回保存的变量。
    //定义一个删除函数，参数为位置i，返回被删除的元素
int delete(int i) {
  //判断i是否合法
  if (i < 1 || i > length) {
    //返回错误信息
    printf("Invalid position!\n");
    return -1;
  }
  //保存被删除的元素
  int x = array[i - 1];
  //从第i+1个元素开始，依次前移一位
  for (int j = i; j < length; j++) {
    array[j - 1] = array[j];
  }
  //将length减一
  length--;
  //返回被删除的元素
  return x;
}

栈

栈是一种只能在一端进行插入和删除的线性表，这一端叫做栈顶，另一端叫做栈底。
栈的插入操作叫做入栈或者压栈，删除操作叫做出栈或者弹栈。
栈遵循先进后出的原则，也就是说最后入栈的元素最先出栈。

//定义一个最大容量为10的数组
#define MAXSIZE 10
int stack[MAXSIZE];
//初始化栈顶为-1
int top = -1;
//入栈操作，将元素x压入栈中
void push(int x) {
  //判断栈是否已满
  if (top == MAXSIZE - 1) {
    printf("Stack is full.\n");
    return;
  }
  //将元素x放入数组中，并将栈顶加一
  stack[++top] = x;
}
//出栈操作，将栈顶元素弹出并返回
int pop() {
  //判断栈是否为空
  if (top == -1) {
    printf("Stack is empty.\n");
    return -1;
  }
  //返回数组中的元素，并将栈顶减一
  return stack[top--];
}

队列

队列是一种只能在一端进行插入操作，在另一端进行删除操作的线性表，插入的一端叫做队尾，删除的一端叫做队头。
队列遵循先进先出的原则，也就是说最先入队的元素最先出队。

//定义一个结点
typedef struct node
{
    int data; //数据域
    struct node *next; //指针域
}Node;

//定义一个队列
typedef struct queue
{
    Node *front; //队头指针
    Node *rear; //队尾指针
}Queue;

//初始化一个队列
void initQueue(Queue *q)
{
    q->front = NULL; //队头指针置空
    q->rear = NULL; //队尾指针置空
}

//判断队列是否为空
int isEmpty(Queue *q)
{
    return q->front == NULL; //如果队头指针为空，说明队列为空，返回1，否则返回0
}

//入队操作，从队尾插入元素
void enQueue(Queue *q, int data)
{
    Node *newNode = (Node *)malloc(sizeof(Node)); //创建一个新结点
    newNode->data = data; //给新结点的数据域赋值
    newNode->next = NULL; //给新结点的指针域赋空值

    if(isEmpty(q)) //如果队列为空，那么新结点既是队头也是队尾
    {
        q->front = newNode;
        q->rear = newNode;
    }
    else //如果队列不为空，那么新结点成为新的队尾，原来的队尾指向新结点
    {
        q->rear->next = newNode;
        q->rear = newNode;
    }
}

//出队操作，从队头删除元素，并返回元素值
int deQueue(Queue *q)
{
    if(isEmpty(q)) //如果队列为空，返回-1表示错误
    {
        return -1;
    }
    else //如果队列不为空，删除队头元素，并返回其值
    {
        Node *temp = q->front; //用一个临时指针保存要删除的结点
        int data = temp->data; //保存要返回的数据

        q->front = q->front->next; //更新队头指针

        if(q->front == NULL) //如果删除后队列为空，那么更新队尾指针为NULL
        {
            q->rear = NULL;
        }

        free(temp); //释放要删除的结点的内存空间

        return data; //返回出队元素的值
    }
}

树

树是一种分层的数据结构，有一个根节点和若干子节点。每个节点可以有零个或多个子节点，每个子节点只有一个父节点。
树可以用来表示层次关系或者分类关系。

//定义一个最大子节点数为10的常量
#define MAX_CHILDREN 10
//定义一个树节点结构体
typedef struct TreeNode {
  int data; //数据域
  struct TreeNode *children[MAX_CHILDREN]; //指针域，存储子节点
} TreeNode;
//创建一个新的树节点，并赋值数据域为x，指针域为NULL
TreeNode *newNode(int x) {
  //分配内存空间
  TreeNode *node = (TreeNode *)malloc(sizeof(TreeNode));
  //赋值数据域
  node->data = x;
  //初始化指针域
  for (int i = 0; i < MAX_CHILDREN; i++) {
    node->children[i] = NULL;
  }
  //返回新节点
  return node;
}
//创建一个示例树，如下图所示
//      1
//    / | \
//   2  3  4
//  / \    |
// 5   6   7
TreeNode *createExampleTree() {
  //创建根节点，并赋值为1
  TreeNode *root = newNode(1);
  //创建第二层的三个子节点，并赋值为2，3，4
  root->children[0] = newNode(2);
  root->children[1] = newNode(3);
  root->children[2] = newNode(4);
  //创建第三层的三个子节点，并赋值为5，6，7
  root->children[0]->children[0] = newNode(5);
  root->children[0]->children[1] = newNode(6);
  root->children[2]->children[0] = newNode(7);
  //返回根节点
  return root;
}

预处理器命令

预处理器指令是以井号（#）开头的一些指令，它们可以用来简化源程序在不同的执行环境中的更改和编译。
C语言的预处理器(CPP)是一个文本替换工具，它会在编译器处理程序之前对源代码进行一些操作，比如包含头文件，扩展宏，进行条件编译等。
它不是编译器的组成部分，而是编译过程中的一个单独的步骤。我们可以用预处理器指令来扩充程序设计的环境。

指令 描述
#define 定义宏
#include 包含一个源代码文件
#undef 取消已定义的宏
#ifdef 如果宏已经定义，则返回真
#ifndef 如果宏没有定义，则返回真
#if 如果给定条件为真，则编译下面代码
#else #if的替代方案
#elif 如果前面的 #if 给定条件不为真，当前条件为真，则编译下面代码
#endif 结束一个 #if……#else 条件编译块
#error 当遇到标准错误时，输出错误消息
#pragma 使用标准化方法，向编译器发布特殊的命令到编译器中

告诉 CPP 把所有的 MAX_ARRAY_LENGTH 定义为 20。使用 #define 定义常量来增强可读性。

#define MAX_ARRAY_LENGTH 20
````

> 告诉 CPP 从本地目录中获取 myheader.h，并添加内容到当前的源文件中。

告诉 CPP 取消已定义的 FILE_SIZE，并定义它为 42

#ifdef DEBUG
   /* Your debugging statements here */

预处理器运算符

  • 宏延续运算符
    一个宏通常写在一个单行上。但是如果宏太长，一个单行容纳不下，则使用宏延续运算符（\）。
  • 字符串常量化运算符
    在宏定义中，当需要把一个宏的参数转换为字符串常量时，则使用字符串常量化运算符（#）。
    #include <stdio.h>
 
#define  message_for(a, b)  \
    printf(#a " and " #b ": We love you!\n")

int main(void)
{
  message_for(Carole, Debra);
  return 0;
}
  • 标记粘贴运算符
    宏定义内的标记粘贴运算符（##）会合并两个参数。它允许在宏定义中两个独立的标记被合并为一个标记。
    #include <stdio.h>
 
#define tokenpaster(n) printf ("token" #n " = %d", token##n)

int main(void)
{
  int token34 = 40;
  
  tokenpaster(34);
  return 0;
}

参数化的宏

使用参数化的宏来模拟函数。
在使用带有参数的宏之前，必须使用 #define 指令定义。参数列表是括在圆括号内，且必须紧跟在宏名称的后边。宏名称和左圆括号之间不允许有空格。

#include <stdio.h>
 
#define MAX(x,y) ((x) > (y) ? (x) : (y))
 
int main(void)
{
   printf("Max between 20 and 10 is %d\n", MAX(10, 20));  
   return 0;
}

可变参数

可变参数函数是指可以接受可变数量和类型的参数的函数，例如 printf（）和 scanf（）。
要定义一个可变参数函数，您需要使用stdarg.h头文件中的一些宏，例如 va_list,va_start,va_arg和va_end。

  • va_start(ap, last_arg)
    初始化可变参数列表。ap 是一个 va_list 类型的变量，last_arg 是最后一个固定参数的名称（也就是可变参数列表之前的参数）。该宏将 ap 指向可变参数列表中的第一个参数。

  • va_arg(ap, type)
    获取可变参数列表中的下一个参数。ap 是一个 va_list 类型的变量，type 是下一个参数的类型。该宏返回类型为 type 的值，并将 ap 指向下一个参数。

  • va_end(ap)
    结束可变参数列表的访问。ap 是一个 va_list 类型的变量。该宏将 ap 置为 NULL。

    #include <stdio.h>
#include <stdarg.h>

// 定义一个可变参数函数，第一个参数是整数个数，后面是任意个整数
double average(int num, ...)
{
    va_list valist; // 定义一个 va_list 类型的变量
    double sum = 0.0;
    int i;

    // 用 va_start 宏初始化 valist 变量，第二个参数是可变参数列表中的第一个参数
    va_start(valist, num);

    // 用 va_arg 宏访问每个可变参数，并累加到 sum 中
    for (i = 0; i < num; i++)
    {
        sum += va_arg(valist, int);
    }

    // 用 va_end 宏清理 valist 变量
    va_end(valist);

    return sum / num;
}

int main()
{
    printf("Average of 2, 3, 4, 5 = %f\n", average(4, 2, 3, 4, 5));
    printf("Average of 5, 10, 15 = %f\n", average(3, 5, 10, 15));
}
